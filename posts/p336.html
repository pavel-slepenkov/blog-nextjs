January 19, 2014

Today I have to install the old version of application which was frozen a half year ago to developer org (because a fresh dev. org have no any dependency while sandbox have all metadata from Production).

The first step for such kind of migration is creation data model, because of all other components have a dependencies on data model. For this app I have to migrate 46 objects (both standard and custom).
